# Three.js Platformer (Mario-like)

Tiny browser platformer inspired by classic platformers.

## Run locally

```bash
npm install
npm run dev
```

Open the URL printed by Vite.

## Controls
- Move: A/D or ←/→
- Jump: Space / W / ↑
- Restart: R
- Click to lock pointer (Esc to unlock)

## Notes
- Simple AABB collision + gravity
- One short level, coins, enemies, and a goal pole

## Deploy (optional)
You can deploy to GitHub Pages by building and serving `dist/`.

```bash
npm run build
```
